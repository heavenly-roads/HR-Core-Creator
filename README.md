This is a development repository, final releases are available at CurseForge: https://www.curseforge.com/hytale/mods/hr-core-creator/files/all

This mod includes original blocks, lighting, decoration, interactive elements, effects and extras for worlds, minigames and servers. This mod will grow as the game evolves to become a huge bundle of elements to boost creativity.

Supported languages: <i>English</i> | <i>Spanish</i>
